# 视频去水印网站

一个简洁高效的视频去水印工具网站，支持多平台视频链接解析和下载。

## 功能特点

- 支持多种链接格式解析：
  - 视频直链接
  - 用户访问链接
  - 用户分享链接
- 提供原视频预览功能
- 支持选择下载分辨率
- 快速解析和下载
- 简洁美观的界面设计
- 响应式布局，支持各种设备

## 技术栈

- HTML5
- CSS3
- JavaScript
- Fetch API
